# Plan: Unified RankedCard Component & Homepage Rewire

## Goal

Build one unified card component used by every homepage section. Rewire the homepage to display 8 ranked sections with the new design. Remove old components (Browse by Category, State rows).

## Tasks

### Task 1: Create RankedCard component

**File:** `src/components/RankedCard.tsx` (new)

**Props:**
```ts
interface RankedCardProps {
  company: Company;
  rank: number;  // 1-based
}
```

**Design specifications:**
- Large italic stroke number (matching current Featured Subcontractors style: `font-display text-[9rem] font-bold leading-[0.8] italic`)
- Number does NOT overlap the company name — place number to the left, card to the right with proper spacing (no `mr-[-30px]`)
- `WebkitTextStroke: '3px #334155'` default, `'3px #F5C518'` on hover (gold)
- Card body: dark gradient (`bg-gradient-to-t from-black via-[#1A1A1A] to-[#2A2A2A]`), portrait aspect ratio (`aspect-[2/3]`), `w-[150px]`
- Card face shows ONLY the company name — `font-display text-[0.75rem] font-bold uppercase text-white`
- No city, no state, no rating, no revenue on the card face
- Card wraps in a `<Link to={/company/${id}}>`
- `scrollSnapAlign: 'start'`

### Task 2: Create RankedCardPopup component

**File:** `src/components/RankedCardPopup.tsx` (new)

**Appears on hover** (desktop only, hidden on mobile):
- City, State displayed
- Favorites heart button (uses `useFavorites` hook)
- Compact design — small floating card near the hovered card
- `opacity-0 group-hover:opacity-100` transition
- Position: below or beside the card, z-indexed above siblings

### Task 3: Create RankedRail component

**File:** `src/components/RankedRail.tsx` (new)

A horizontal scroll rail specifically for ranked sections.

**Props:**
```ts
interface RankedRailProps {
  title: string;          // e.g., "Top 25 General Contractors"
  companies: Company[];
  isLoading?: boolean;
}
```

**Design:**
- Gold accent bar + title (matching current ContentRail style)
- Exactly 5 cards visible at a time — card width calculated as `calc((100% - gaps - arrow-space) / 5)`
- Scroll arrows at far right edge (white bg, gray border, matching current ScrollArrows pattern)
- `overflow-x-auto scrollbar-hide` with `scroll-snap-type: x mandatory`
- Loading skeleton: 5 placeholder cards
