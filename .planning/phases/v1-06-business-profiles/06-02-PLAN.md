# Plan 06-02: Profile Editing UI

## Goal
Verified businesses can edit their profile through a self-service dashboard with media uploads. Non-verified businesses see a clear upgrade path.

## Task 1: Edit Profile page + media upload

### Create src/pages/EditProfile.tsx
Self-service profile editing dashboard:
- Route: `/company/:id/edit`
- On mount: check auth (useAuth), fetch company data, check verificationStatus
- If not verified: redirect to company profile or show access denied
- Sections:
  1. **Contact Info** — editable fields: phone, email, website, address, zipCode
  2. **Services** — tag-style editor: list of service strings, add/remove
  3. **Business Details** — warranty (text input), emergencyService (toggle), freeEstimate (toggle)
  4. **Portfolio Photos** — grid of uploaded photos + upload button (drag-and-drop)
  5. **Videos** — list of uploaded videos + upload button

- Each section has a "Save" button that calls `ProfileAPI.update(companyId, { ...changes })`
- Changes visible immediately (PROF-04) — update local state on successful save
- Toast/notification on success: "Profile updated successfully"

### Photo upload flow (in EditProfile.tsx)
1. User clicks "Upload Photo" or drops file on drop zone
2. Validate: file.type in image/jpeg|png|webp, file.size <= 10MB
3. Call `ProfileAPI.getUploadUrl(companyId, { filename, contentType, fileSize })`
4. Call `ProfileAPI.uploadFile(uploadUrl, file)` — direct PUT to R2
5. Call `ProfileAPI.registerMedia(companyId, { key, type: 'photo', filename, fileSize })`
6. Add to local media list, show thumbnail

### Video upload flow (same pattern, max 100MB, video/mp4|webm)

### Styling
- Dark theme consistent with rest of app (bg-background, bg-surface, border-border)
- Form inputs: `bg-background border border-border rounded px-3 py-2 text-white focus:border-brand-primary outline-none`
- Save buttons: `bg-brand-primary text-white font-display uppercase tracking-wider`
- Upload zone: dashed border, drag hover state with brand-primary highlight

### Create src/components/MediaUploader.tsx
Reusable upload component:
- Props: `{ companyId, type: 'photo'|'video', onUpload: (media: MediaRecord) => void, maxSize: number, accept: string[] }`
- Drag-and-drop zone + click-to-select
- Upload progress indicator (use onUploadProgress or fake progress)
- Error display for invalid files

### Create src/components/MediaGallery.tsx
Grid display of uploaded media:
- Props: `{ media: MediaRecord[], onDelete?: (id: string) => void, editable?: boolean }`
- Photos: 3-column grid with hover overlay showing delete button (if editable)
- Videos: list with thumbnail placeholder + filename + delete button
- Empty state: "No photos yet — upload your first portfolio photo"
