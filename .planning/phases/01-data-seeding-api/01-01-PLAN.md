### Task 2: Create top-companies API endpoint

**File:** `server/routes/companies.ts` (add to existing)

Add endpoint: `GET /api/top-companies`

**Query params:**
- `category` (required) — category name to filter by
- `subCategory` (optional) — for subcategory filtering (e.g., "Masonry/Concrete")
- `limit` (optional, default 25) — max results

**Logic:**
```sql
SELECT id, businessName, city, state, rating, reviewCount, category, subCategory
FROM companies
WHERE category = :category
  OR subCategory = :subCategory
ORDER BY rating DESC, reviewCount DESC
LIMIT :limit
```

**Response format:**
```json
{
  "companies": [...],
  "total": 25,
  "category": "Electrical"
}
```
